Smart  Grid is a super responsive and very advance wordpress Plugin. You can Display Post, WooCommerce Produce and Portfolio as Grid View with this PLugin. 

== Description ==
Smart  Grid is a super responsive and very advance wordpress Plugin . It's has a nice plugin option. You can easily controle it from plugin option. 

<strong>Features:-</strong>
<ul>
 	<li>Display you blog post</li>
 	<li>Display Your  Woocommerce Product</li>
 	<li>DIsplay Your Portfolio</li>
 	<li>Nice Plugin Option</li>
 	<li>Easy To Customise </li>
 	<li>All Shortcode Available</li>
 	<li>and more feature</li>
</ul>

  How to use this plugin:-

<pre>Use " [smart_grid_post]" this shortcode  to display your post  as grid view</pre>
<pre>Use " [smart_grid_product]" this shortcode  to display your WooCommerce Product as grid view</pre>
<pre>Use " [smart_grid_portfolio]" this shortcode  to display your Portfolio grid view</pre>
<pre>You can also easily select page template from page attribute to display anygrid </pre>
